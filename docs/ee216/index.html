EE216 Lab - Probability Statistics and Numerical Methods ​

Many engineering problems have inherent uncertainties; hence, it is important to learn tools from probability and statistics to understand, model, and manage these uncertainties.

Topics covered in this course include:

  • Combinatorial analysis
  • Axioms of probability
  • Conditional probability and independence
  • Random variables
  • Descriptive statistics
  • Sampling distribution
  • Tests of hypotheses based on small sample sizes
  • Regression
  • Numerical solutions of non-linear and differential equations

There will be an emphasis on applying these tools using computer-based methods (using Python and JupyterLab).

1. Link to Canvas ​

Go to EE216 Lab Canvas page https://canvas.usask.ca/courses/117232 for schedule and lab submissions.

2. Install Jupyter Lab ​

  1. Install Python (e.g., version 3.13.11 from https://www.python.org/downloads/windows/)
  2. Install JuypterLab (info from https://jupyter.org/install):
bash
pip install jupyterlab
  1. Once installed, launch JupyterLab with an appropriate work directory:
bash
jupyter lab --notebook-dir=d:/Dropbox/uofs/Classes/EE216/EE216_2026/jupyter

Further documentation can be found at https://jupyterlab.readthedocs.io/en/latest/
