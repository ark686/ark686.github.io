Lab 3 - Axioms of Probability ​

1. Overview ​

The Three Axioms of Probability:

  • Axiom 1 (Non-negativity):

    • For any Event "A", P(A) >= 0
    • Probabilities cannot be negative.

  • Axiom 2 (Normalization):

    • The probability of the sample space is one, P(S) = 1
    • Something must happen, this establishes that probabilities range from 0 to 1

  • Axiom 3 (Countable Additivity):

    • For any countable sequence of mutually exclusive events A1, A2, A3, ..., P(Union of As) = Sum(P(As))
    • If events cannot occur simultaneously (mutually exclusive), the probability that at least one occurs is the sum of individual probabilities.

2. Examples - Simulating Random Experiments ​

The key idea: if we cannot compute P(A) analytically, we can estimate it by:

  1. Simulating the experiment many times (n trials)
  2. Counting how often event A occurs: n(A)
  3. Estimating: Pˆ(A) = n(A)/n By the Law of Large Numbers, Pˆ(A) → P(A) as n → ∞

2.2 Monte Carlo - Sum of Two Dice ​

python
import numpy as np
from numpy.random import default_rng

def monte_carlo_dice_sum(n_simulations =100000):
    """
    Estimate P(sum of two dice = 7) via Monte Carlo.
    True probability = 6/36 = 1/6 = 0.1667 (approx)
    """
    rng = default_rng(seed=42)

    # Simulate n_simulations rolls of two dice
    die1 = rng.integers(1, 7, size=n_simulations)
    die2 = rng.integers(1, 7, size=n_simulations)

    # Calculate sums
    sums = die1 + die2

    # Count how many times sum equals 7
    count_7 = np.sum(sums == 7)

    # Estimate probability
    prob_estimate = count_7 / n_simulations

    return prob_estimate

# Run simulation
estimate = monte_carlo_dice_sum(n_simulations =100000)
true_prob = 6/36
print(f"Monte Carlo estimate: {estimate :.4f}")
print(f"True probability: {true_prob :.4f}")
print(f"Error: {abs(estimate - true_prob):.4f}")

2.3 Monte Carlo - Birthday Problem ​

python
import numpy as np
from numpy.random import default_rng

def birthday_simulation(k, n_simulations =10000):
    """
    Estimate P(at least two people share a birthday)
    in a group of k people.
    """
    rng = default_rng(seed=42)
    shared_count = 0

    for _ in range(n_simulations):
       # Generate k random birthdays (1-365)
       birthdays = rng.integers(1, 366, size=k)

       # Check if any birthday appears more than once
       if len(birthdays) != len(set(birthdays)):
           shared_count += 1

    return shared_count / n_simulations

# Test for various group sizes
print("k\tMonte Carlo\tExact")
print("-" * 35)
for k in [10, 20, 23, 30, 50]:
    mc_prob = birthday_simulation(k, n_simulations =50000)
    # Exact calculation using complement
    exact_prob = 1 - np.prod([(365-i)/365 for i in range(k)])
    print(f"{k}\t{mc_prob :.4f}\t\t{exact_prob :.4f}")

2.4 Law of Large Numbers Visualization ​

python
import numpy as np
import matplotlib.pyplot as plt
from numpy.random import default_rng

def plot_lln_convergence():
    """
    Visualize how relative frequency converges to true probability.
    Example: Fair coin with P(Heads) = 0.5
    """
    rng = default_rng(seed=42)
    n_flips = 10000

    # Simulate coin flips (1 = Heads , 0 = Tails)
    flips = rng.integers(0, 2, size=n_flips)

    # Calculate running proportion of heads
    cumsum = np.cumsum(flips)
    n_values = np.arange(1, n_flips + 1)
    proportions = cumsum / n_values

    # Plot
    plt.figure(figsize=(10, 5))
    plt.plot(n_values , proportions , 'b-', alpha=0.7, linewidth =0.5)
    plt.axhline(y=0.5, color='r', linestyle='--', label='True P(Heads) = 0.5')
    plt.xlabel('Number of Flips (n)')
    plt.ylabel('Proportion of Heads')
    plt.title('Law of Large Numbers: Relative Frequency Convergence ')
    plt.legend()
    plt.xscale('log')
    plt.grid(True , alpha =0.3)
    plt.tight_layout()
    plt.savefig('lln_convergence.png', dpi=150)
    plt.show()

plot_lln_convergence()
