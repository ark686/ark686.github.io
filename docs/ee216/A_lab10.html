2. Introduction ​

Let A be an n × n square matrix. A scalar λ is an eigenvalue of A if there exists a nonzero vector x such that:

Ax = λx

The nonzero vector x is called an eigenvector corresponding to the eigenvalue λ.

The prefix “eigen” comes from German, meaning “own” or “characteristic.” Eigenvalues are also called characteristic values, proper values, or latent roots.

The eigenvalue equation Ax = λx has a beautiful geometric interpretation:

When a matrix A multiplies a general vector, it typically changes both the direction and magnitude of that vector. However, eigenvectors are special: they only get scaled (stretched or compressed) by their matrix, not rotated.

  • If λ > 1: The eigenvector is stretched
  • If 0 < λ < 1: The eigenvector is compressed
  • If λ < 0: The eigenvector is reversed and scaled
  • If λ = 1: The eigenvector is unchanged
  • If λ = 0: The eigenvector is mapped to zero (matrix is singular)

Eigenvectors represent the “natural directions” of a linear transformation.

Eigenvalues and eigenvectors are among the most important concepts in applied mathematics and engineering. They appear throughout electrical and computer engineering in contexts such as:

  • Control Systems: Eigenvalues determine the stability of linear dynamical systems
  • Signal Processing: Principal Component Analysis (PCA) uses eigenvalue decomposition for dimensionality reduction
  • Vibration Analysis: Natural frequencies of mechanical and electrical systems are eigenvalues
  • Quantum Computing: Quantum states are described by eigenvectors of Hermitian opera- tors
  • Power Systems: Modal analysis of large interconnected networks uses eigenvalue methods
  • Machine Learning: Many algorithms, including spectral clustering and recommendation systems, rely on eigenvalue decomposition

    [-20, 40, -20],

3.3 Implementing the Power Method ​

python
import numpy as np

def power_method(A, tol =1e-8, max_iter =100):
    """
    Find the dominant eigenvalue and eigenvector using the power method .

    Parameters:
    A: Square matrix(numpy array)
    tol: Convergence tolerance
    max_iter: Maximum iterations

    Returns:
    eigenvalue: Dominant eigenvalue
    eigenvector: Corresponding unit eigenvector
    iterations: Number of iterations to converge
    """
    n = A.shape[0]
    x = np.ones(n) # Initial guess
    x = x / np.linalg.norm(x) # Normalize

    eigenvalue_old = 0

    for k in range(max_iter):
        # Matrix -vector multiplication
        y = A @ x

        # Compute Rayleigh quotient for eigenvalue estimate
        eigenvalue = np.dot(x, y)

        # Normalize
        x = y / np.linalg.norm(y)

        # Check convergence
        if abs(eigenvalue - eigenvalue_old) < tol:
            return eigenvalue, x, k + 1

        eigenvalue_old = eigenvalue
    return eigenvalue, x, max_iter

# Test on example matrix
A = np.array([[40, -20, 0],
    [-20, 40, -20],
    [0, -20, 40]], dtype=float)

eigenvalue, eigenvector, iters = power_method(A)
print(f"Dominant eigenvalue: {eigenvalue:.6f}")
print(f"Eigenvector: {eigenvector}")
print(f"Converged in {iters} iterations")

# Compare with numpy
vals, vecs = np.linalg.eig(A)
print(f"\nNumPy max eigenvalue: {np.max(vals):.6f}")

3.4 Visualization of Eigenvectors ​

python
import numpy as np
import matplotlib .pyplot as plt

# 2x2 matrix for easy visualization
A= np.array([[2, 1],
    [1, 2]])

eigenvalues, eigenvectors= np.linalg.eig(A)
print(f"Eigenvalues: {eigenvalues}") #[3, 1]
print(f"Eigenvectors(columns):\n{eigenvectors}")

# Create figure with two subplots
fig, axes= plt.subplots(1, 2, figsize=(14, 6))

# LEFT PLOT:Show eigenvectors and their transformations(scaling only)
ax1= axes[0]

for i,(lam, color) in enumerate(zip(eigenvalues,['blue', 'red'])):
    v= eigenvectors[:, i]
    Av= A @ v # This equals lambda * v

    # Original eigenvector(unit length from numpy)
    ax1.arrow(0, 0, v[0], v[1], head_width=0.08, head_length=0.05,
        fc=color, ec=color, linewidth=2, alpha=0.7,
        label=f'$v_{i+1}$(original)')

    # Transformed eigenvector= lambda * v(just scaled !)
    ax1.arrow(0, 0, Av[0], Av[1], head_width=0.08, head_length=0.05,
        fc=color, ec=color, linewidth=2, linestyle='--',
        label=f'$Av_{i+1}= {lam:.1f} \\ cdot v_{i+1}$')

    # Draw dashed line showing eigenvector direction extends
    t= np.linspace(-2, 2, 100)
    ax1.plot(t*v[0], t*v[1], '--', color=color, alpha=0.3, linewidth=1)

ax1.set_xlim(-2.5, 2.5)
ax1.set_ylim(-2.5, 2.5)
ax1.set_aspect('equal')
ax1.grid(True, alpha=0.3)
ax1.axhline(y=0, color='k', linewidth=0.5)
ax1.axvline(x=0, color='k', linewidth=0.5)
ax1.legend(loc='upper left', fontsize=9)
ax1.set_title('Eigenvectors:A only SCALES them\n(solid= original, dashed= transformed)')
ax1.set_xlabel('$x_1$')
ax1.set_ylabel('$x_2$')

# RIGHT PLOT:Compare with a NON-eigenvector(gets rotated AND scaled)
ax2= axes[1]

# Plot an eigenvector(blue)
v1= eigenvectors[:, 0]
Av1= A @ v1
ax2.arrow(0, 0, v1[0], v1[1], head_width=0.08, head_length=0.05,
    fc='blue', ec='blue', linewidth=2, label='Eigenvector $v_1$')
ax2.arrow(0, 0, Av1[0], Av1[1], head_width=0.08, head_length=0.05,
    fc='blue', ec='blue', linewidth=2, linestyle='--',
    label=f'$Av_1$(scaled by $\\ lambda_1={eigenvalues[0]:.1f}$)')

# Plot a non-eigenvector(green)-this will be rotated
w= np.array([1, 0]) # Not an eigenvector
Aw= A @ w
ax2.arrow(0, 0, w[0], w[1], head_width=0.08, head_length=0.05,
    fc='green', ec='green', linewidth=2, label='Non-eigenvector $w$')
ax2.arrow(0, 0, Aw[0], Aw[1], head_width=0.08, head_length=0.05,
    fc='green', ec='green', linewidth=2, linestyle='--',
    label='$Aw$(scaled AND rotated !)')

ax2.set_xlim(-0.5, 3)
ax2.set_ylim(-0.5, 2.5)
ax2.set_aspect('equal')
ax2.grid(True, alpha=0.3)
ax2.axhline(y=0, color='k', linewidth=0.5)
ax2.axvline(x=0, color='k', linewidth=0.5)
ax2.legend(loc='upper left', fontsize=9)
ax2.set_title('Eigenvector vs Non-eigenvector \n(non-eigenvectors change direction)')
ax2.set_xlabel('$x_1$')
ax2.set_ylabel('$x_2$')

plt.tight_layout()
plt.savefig('eigenvector_scaling .png', dpi=150)
plt.show()

# Verify numerically
print("\nVerification:")
print(f"v1= {v1}")
print(f"A @ v1= {Av1}")
print(f"lambda1 * v1= {eigenvalues[0] * v1}")
print(f"Difference:{np.linalg .norm(Av1-eigenvalues[0]* v1):.2e}")

3.5 ECE Application: Stability Analysis ​

python
import numpy as np
import matplotlib .pyplot as plt

def analyze_stability(A, system_name="System"):
    """
    Analyze stability of dx/dt= Ax based on eigenvalues .
    """
    eigenvalues= np.linalg.eigvals(A)

    print(f"\n{system_name} Analysis:")
    print("-"* 40)
    print(f"Eigenvalues: {eigenvalues}")

    # Check stability
    real_parts= np.real(eigenvalues)

    if all(real_parts < 0):
        stability="STABLE(all eigenvalues have negative real parts)"
    elif any(real_parts > 0):
        stability="UNSTABLE(at least one eigenvalue has positive real part)"
    else:
        stability="MARGINALLY STABLE(eigenvalues on imaginary axis)"

    print(f"Stability: {stability}")

    # Plot eigenvalues in complex plane
    plt.figure(figsize=(8, 6))
    plt.scatter(np.real(eigenvalues), np.imag(eigenvalues),
        s=100 , c='red', marker='x', linewidths=2)

    # Shade stable region
    plt.axvspan(-3, 0, alpha=0.2 , color='green', label='Stable region')
    plt.axvspan(0, 3, alpha=0.2 , color='red', label='Unstable region')

    plt.axhline(y=0, color='k', linewidth=0.5)
    plt.axvline(x=0, color='k', linewidth=0.5)
    plt.xlabel('Real Part')
    plt.ylabel('Imaginary Part')
    plt.title(f'{system_name} - Eigenvalue Plot')
    plt.legend()
    plt.grid(True , alpha=0.3)
    plt.tight_layout()
    plt.savefig(f'{system_name .lower().replace("",
        "_")} _stability .png', dpi=150)
    plt.show()

    return eigenvalues

# Example 1: Stable RLC circuit(overdamped)
A_stable= np.array([[0 , 4],
    [-1, -4]])
analyze_stability(A_stable ,"Overdamped RLC Circuit")

# Example 2: Oscillatory system
A_oscillatory= np.array([[0 , 1],
    [-4, 0]])
analyze_stability(A_oscillatory ,"Undamped Oscillator")

# Example 3: Unstable system
A_unstable= np.array([[1 , 2],
    [0, 3]])
analyze_stability(A_unstable, "Unstable System")

3.6 Markov Chain Steady State ​

python
import numpy as np

def find_steady_state(P):
    """
    Find the steady -state distribution of a Markov chain.
    The steady state is the eigenvector for eigenvalue = 1.
    """
    eigenvalues, eigenvectors = np.linalg.eig(P)
    # Find index of eigenvalue closest to 1
    idx = np.argmin(np.abs(eigenvalues - 1.0))

    # Get corresponding eigenvector
    steady_state = np.real(eigenvectors[:, idx])

    # Normalize so elements sum to 1(probability distribution)
    steady_state = steady_state / np.sum(steady_state)

    return steady_state

# Network router state transition matrix
# States : Idle, Low traffic, High traffic
P = np.array([[0.7, 0.2, 0.1],
    [0.2, 0.6, 0.3],
    [0.1, 0.2, 0.6]])

steady_state = find_steady_state(P)
state_names =['Idle', 'Low', 'High']

print("Markov Chain Steady -State Analysis ")
print("=" * 40)
print("\nTransition Matrix P:")
print(P)
print("\nSteady-state distribution :")
for name, prob in zip(state_names, steady_state):
    print(f" {name}: {prob :.4f}({ prob *100:.1f}%)")

# Verify : P @ pi = pi
print("\nVerification(P @ pi should equal pi):")
print(f"P @ pi = {P @ steady_state}")
print(f"pi = { steady_state}")

5. Exercise 10.2 ​

Consider the following circuit:

We want to know the voltages at nodes 1, 2, 3, and 4. One way of calculating these values is nodal analysis.

First we construct a "Nodal Conductance Matrix" G (4 x 4 as four nodes):

  • Diagonal Gii: sum of ALL conductances connected to node i (includes both shunt branches to ground and series branches to other nodes)
  • Off-diagonal Gij (i≠j): negative sum of conductances directly between nodes i and j; zero if no direct branch exists

Remember that conductance is the inverse of resistance.

We can directly solve for the voltages using:

G×v=I

where:

  • G is the nodal conductance matrix
  • v is a 4-element array representing the voltages at the four nodes
  • I is a 4-element array represent the current "injected" into each of the four nodes (i.e., I=[2.0,0.0,0.0,0.0])

We can also solve for the voltages using eigenvectors:

  1. Determine the eigenvalues and eigenvectors of nodal conductance matrix G
  2. Using "modal decomposition":
G×v=I→v=V(VᵀI/λ)

where:

  • V is the array of eigenvectors
  • I is the array of injected current
  • λ is the array of eigenvalues

Write a Python program to:

  1. Determine the four node voltages using the direct solution
  2. Determine the four node voltages using eigenvectors
  3. Compare the two approaches and confirm they give the same results
