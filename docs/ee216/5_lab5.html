1. Random Variables ​

A random variable is a function of X : S → R that assigns a real number to each outcome in the sample space. We typically denote random variables by uppercase letters (X, Y, Z) and their values by lowercase letters (x, y, z).

Think of a random variable as a “measurement device” applied to random outcomes. The experiment produces an outcome ω ∈ S, and the random variable X produces the number X(ω). Different outcomes may produce the same number.

Example: Coin Tossing

Consider tossing a fair coin three times. The sample space is:

S = {HHH, HHT, HTH, HTT, THH, THT, TTH, TTT}

Define X = number of heads. Then X is a random variable with:

X(HHH) = 3, X(HHT) = X(HTH) = X(THH) = 2  
X(HTT) = X(THT) = X(TTH) = 1, X(TTT) = 0

The random variable X takes values in {0, 1, 2, 3}, converting the original sample space of sequences into numbers.

Discrete versus Continuous Random Variables

A random variable X is discrete if it can take on only a finite or countably infinite number of distinct values. Common examples include counts, rankings, and categorical encodings.

A random variable X is continuous if its set of possible values forms an interval (or union of intervals) on the real line, and the probability of any single value is zero: P(X = x) = 0 for all x.

2. Probability Mass Function (PMF) ​

The probability distribution of a discrete random variable is completely characterized by its probability mass function.

The probability mass function (PMF) of a discrete random variable X is the function p : R → [0, 1] defined by:

p(x) = P(X = x)

for all x ∈ R. The PMF gives the probability that X takes each possible value.

Example: PMF of a Fair Die

Roll a fair six-sided die. Let X = the number showing. The PMF is:

p(x) = 1/6 is x ∈ {1, 2, 3, 4, 5, 6}, 0 otherwise

The total of all probabilities = 1/6 x 6 = 1

Example: Three Coin Tosses

Toss a fair coin 3 times and let X = number of heads. To find the PMF, we count outcomes:

Number of heads:

0 TTT 1/8
1 HTT, THT, TTH 3/8
2 HHT, HTH, THH 3/8
3 HHH 1/8

Sums up to 1 as required = 1/8 + 3/8 + 3/8 + 1/8

3. Cumulative Distribution Function (CDF) ​

An alternative way to describe the distribution of a random variable is through its cumulative distribution function.

The cumulative distribution function (CDF) of a random variable X is the function F : R → [0, 1] defined by:

F(x) = P(X ≤ x)

for all x ∈ R. The CDF gives the probability that X is at most x.

For discrete random variables, the CDF is a step function that jumps at each possible value.

Example: CDF of Die Roll

For X = outcome of rolling a fair die, the CDF is:

F(x) = 0   if x < 1
       1/6 if 1 ≤ x < 2
       2/6 if 2 ≤ x < 3
       3/6 if 3 ≤ x < 4
       4/6 if 4 ≤ x < 5
       5/6 if 5 ≤ x < 6
       1   if x ≥ 6

4. Expected Value ​

The expected value (or mean, or expectation) of a discrete random variable X with PMF p(x) is:

E[X] = μ = μX = ∑(x · p(x))

Example: Expected Value of Die Roll

For a fair die roll:

E[X] = 1/6 x (1 + 2 + 3 + 4 + 5 + 6) = 21 / 6 = 3.5

Note that 3.5 is not a possible outcome—the expected value need not be a possible value of the random variable.

5. Variance and Standard Deviation ​

The variance of a random variable X is:

Var(X) = σ^2 = E[(X - μ)^2]

where μ = E[X]

The standard deviation of a random variable X is:

SD(X)= σ = sqrt(Var(X))

The standard deviation has the same units as X, making it more interpretable than variance.

6. Common Discrete Distributions ​

6.1 Bernoulli Distribution ​

A random variable X has a Bernoulli distribution with parameter p ∈ [0, 1], written X ∼ Bernoulli(p), if X takes only two values:

Compactly: p(x) = p^x * (1-p)^(1-x) for x ∈ {0, 1}.

If X ∼ Bernoulli(p), then:

E[X] = p
Var(X) = p(1 − p) = pq

Example: Bit Error

Each bit transmitted over a noisy channel has probability p = 0.05 of being received in error. Let X = 1 if the bit is in error, X = 0 otherwise. Then X ∼ Bernoulli(0.05) with E[X] = 0.05 and Var(X) = 0.05 · 0.95 = 0.0475.

6.2 Binomial Distribution ​

The binomial distribution counts successes in a fixed number of independent trials.

A random variable X has a binomial distribution with parameters n (number of trials) and p (success probability), written X ∼ Binomial(n, p), if:

If X ∼ Binomial(n, p), then:

E[X] = np
Var(X) = np(1 − p) = npq

Example: Quality Control

A semiconductor fabrication process produces chips with defect probability p = 0.02. A sample of n = 50 chips is tested. Let X = number of defective chips in the sample.

Then X ∼ Binomial(50, 0.02) with:

E[X] = 50 · 0.02 = 1 defective chip expected
Var(X) = 50 · 0.02 · 0.98 = 0.98
SD(x) = sqrt(0.98) ≈ 0.99

6.3 Geometric Distribution ​

The geometric distribution models the number of trials until the first success.

A random variable X has a geometric distribution with parameter p ∈ (0, 1], written X ∼ Geometric(p), if:

Here X represents the number of trials needed to achieve the first success.

If X ∼ Geometric(p) (trials until first success), then:

E(X) = 1/p
Var(X) = (1 - p) / p^2

Example: Packet Retransmission

In a wireless network, each packet transmission succeeds with probability p = 0.9 indepen- dently. Let X = number of transmissions until a packet is successfully delivered.

Then X ∼ Geometric(0.9):

E[X] = 1/0.9 ≈ 1.11 transmissions on average
Var(X) = 0.1/0.81 ≈ 0.123, so SD(X) ≈ 0.35
P(X > 3) = (1 − 0.9)3 = 0.001 (very unlikely to need more than 3 attempts)

6.4 Poisson Distribution ​

The Poisson distribution models rare events occurring in a fixed interval.

A random variable X has a Poisson distribution with parameter λ > 0, written X ∼ Poisson(λ), if:

If X ∼ Poisson(λ), then:

E[X] = λ
Var(X) = λ

The parameter λ is both the mean and variance—a unique property of the Poisson distribution.

The Poisson distribution is appropriate when:

  1. Events occur independently at a constant average rate λ per unit time/space
  2. The probability of two or more events in a small interval is negligible
  3. Events are “rare” relative to the observation window Examples: radioactive decay, network packet arrivals, manufacturing defects, cosmic ray hits

Example: Network Traffic

A web server receives an average of λ = 5 requests per second. Assuming Poisson arrivals:

P(no requests in 1 sec) = (e^(-5) * 5^0) / 0! = e^(-5) ≈ 0.0067
E(X) = 5, SD(X) = sqrt(5) ≈ 2.24

7. Expected Value of Sums ​

7.1 Linearity of Expectation (General Case) ​

For any random variables X and Y (not necessarily independent):

E(X + Y) = E(X) + E(Y)

7.2 Variance of Sums ​

For any random variables X and Y:

Var(X + Y) = Var(X) + Var(Y) + 2Cov(X, Y)

where Cov(X, Y) = E[(X − μX)(Y − μY)] = E[XY] − E[X]E[Y] is the covariance.

If X and Y are independent, then Cov(X, Y ) = 0, so:

Var(X + Y) = Var(X) + Var(Y) (independent case)

8.2 Working with PMFs and CDFs ​

Binomial Distribution:

python
# Define binomial distribution : n=10 trials , p=0.3 success prob
n, p = 10, 0.3
X = stats.binom(n, p)

# PMF: P(X = k)
k = np.arange(0, n+1)
pmf_values = X.pmf(k)
print(f"P(X = 3) = {X.pmf(3):.4f}") # 0.2668

# CDF: P(X <= k)
print(f"P(X <= 3) = {X.cdf(3):.4f}") # 0.6496

# Mean and variance
print(f"E[X] = {X.mean():.4f}") # 3.0
print(f"Var(X) = {X.var():.4f}") # 2.1

# Generate random samples
samples = X.rvs(size =10000)
print(f"Sample mean: { samples.mean():.4f}")
print(f"Sample variance : { samples.var():.4f}")

8.6 Application Error Detection ​

Error Detection Simulation:

python
def simulate_transmission (n_bits , bit_error_prob , n_trials =10000) :
    """
    Simulate bit errors in transmission .
    Returns distribution of error counts .
    """
    # Generate error counts using binomial
    errors = stats.binom.rvs(n=n_bits, p=bit_error_prob, size=n_trials)
    return errors

# Simulate 8-bit transmission with 5% error rate
n_bits = 8
p_error = 0.05
errors = simulate_transmission(n_bits, p_error)

# Analyze results
print(f"Mean errors : {errors.mean():.3f} (theory:{n_bits * p_error})")
print(f"Std errors : {errors.std():.3f}")
print(f"P(no errors): {(errors == 0).mean():.4f} (theory:{(1- p_error)** n_bits :.4f})")
print(f"P(2+ errors): {(errors >= 2).mean():.4f}")

# For Hamming code: can correct 1 error
print(f"P(correctable):{(errors <= 1).mean():.4f}")
