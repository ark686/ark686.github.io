1. Gazebo - Attaching Two Robots ​

In a Gazebo simulation, if you land a Crazyflie robot on top of a TurtleBot robot and then move the TurtleBot, the Crazyflie will not move with the TurtleBot. To fix this, the two robots need to be "attached".

One way fo doing this is to use the Gazabo plugin "gz-sim-detachable-joint-system":

  1. In the TurtleBot URDF description, add the following (with the appropriate "parent_link", "child_model", and "child_link" specified):
xml
<gazebo>
    <plugin filename="gz-sim-detachable-joint-system"
        name="gz::sim::systems::DetachableJoint">
        <!-- The link on the robot -->
        <parent_link>base_link</parent_link>
        <!-- The model to grab -->
        <child_model>crazyflie</child_model>
        <!-- The link on the object -->
        <child_link>crazyflie/base_link</child_link>
        <!-- Topic to detach -->
        <detach_topic>/model/robot/detach</detach_topic>
        <!-- Topic to re-attach -->
        <attach_topic>/model/robot/attach</attach_topic>
    </plugin>
</gazebo>

2. Gazebo - Dynamically Adding Models ​

To dynamically add (or change) a model in a running Gazebo simulation, the Gazebo service /world/<world_name>/create (where <world_name> is replaced with the world name specified in your main "world.sdf" file) can be used.

For example, one way of doing this using Python is:

python
import os

color = 'red'
insert_command = (
    'gz service -s /world/ee464/create'
    '   --reqtype gz.msgs.EntityFactory'
    '   --reptype gz.msgs.Boolean'
    '   --timeout 5000'
    '   --req \''
    '       sdf: "'
    '           <?xml version=\\"1.0\\"?>'
    '           <sdf version=\\"1.4\\">'
    '               <include>'
    '                   <name>stoplight</name>'
   f"                   <uri>./world/{color}.sdf</uri>"
    '               </include>'
    '           </sdf>'
    '       "'
    '       pose: {position: {x:3.1 , y:4.5, z: 0.0}, orientation: {x:0, y:0, z:0, w:1}}'
    '       name: "stoplight"'
    '       allow_renaming: true\''
)
os.system(insert_command)

3. Traffic Light Controller ​

The Quanser Traffic Light modules (shown below) have been extended so that they can be controlled via MQTT.

Use the following code to create a Python script (e.g., "tlset.py"):

python
#**********************************************************************
# tlset.py
#----------------------------------------------------------------------
# 2025-01-03 Andrew Kostiuk
# (c) 2025 University of Saskatchewan
#----------------------------------------------------------------------

import time
import datetime
import sys
import random
import json

from paho.mqtt import client as mqtt_client

MQTT_LOG = 'log'
MQTT_TRAFFIC_LIGHT = 'traffic_light'

class g:
    version = 'V1.01'
    app_name = 'tlset'
    use_system_log = False

    mqtt = None

######################################################################
# Mqtt
#---------------------------------------------------------------------
class Mqtt():
    def __init__(self, app_name, log_function, subscribe_topics, subscribe_handler):
        try:
            self.log_function = log_function

            self.mqttUsername = 'ee464'
            self.mqttPassword = 'badapple2025'
            #self.mqttIp = '127.0.0.1'
            #self.mqttIp = '192.168.129.60'
            self.mqttIp = 'ece2.arkcom.ca'
            self.mqttPort = 1883
            self.ready = False

            self.mqttClientId = f"{app_name}_{random.randint(1000, 9999)}"
            self.mqtt = mqtt_client.Client(self.mqttClientId)
            self.mqtt.username_pw_set(self.mqttUsername, self.mqttPassword)
            self.mqtt.on_connect = self.on_connect
            self.mqtt.connect(self.mqttIp, self.mqttPort)
            if subscribe_topics != None:
                self.subscribe_handler = subscribe_handler
                self.mqtt.subscribe(subscribe_topics)
                self.mqtt.on_message = self.on_message
            self.mqtt.loop_start()
            self.ready = True
        except Exception as e:
            exc_type, exc_obj, exc_tb = sys.exc_info()
            self.log_function(f"EXCEPTION Mqtt: {e} Line {exc_tb.tb_lineno}")

    def ready(self):
        return self.ready

    def on_connect(self, client, userdata, flags, rc):
        if rc == 0:
            self.log_function(f"Mqtt.on_connect: Connected to MQTT Broker {self.mqttIp} ({self.mqttClientId})")
        else:
            self.log_function(f"Mqtt.on_connect: Failed to connect {self.mqttIp}, return code {rc}")

    def on_message(self, client, userdata, msg):
        try:
            if not msg.payload:
                payload = None
            else:
                payload = json.loads(msg.payload)

            if self.subscribe_handler != None:
                self.subscribe_handler(msg.topic, payload)
        except Exception as e:
            exc_type, exc_obj, exc_tb = sys.exc_info()
            self.log_function(f"EXCEPTION Mqtt.on_message: {e} Line {exc_tb.tb_lineno}")

    def publish(self, topic, msg):
        try:
            if self.mqtt == None:
                #MyPrint(f"SystemLog: Aborted log '{log}'")
                return
            result = self.mqtt.publish(topic, json.dumps(msg))
            status = result[0]
            if status != 0:
                self.log_function("Mqtt.publish: Failed to send log")
        except Exception as e:
            exc_type, exc_obj, exc_tb = sys.exc_info()
            self.log_function(f"EXCEPTION Mqtt.publish: {e} Line {exc_tb.tb_lineno}")

#=====================================================================
# SystemLog
#----------------------------------------------------------------------
def SystemLog(log):
    if g.mqtt != None:
        msg = {
            "timestamp": DateAsString(),
            "client":    g.app_name,
            "log":       log
        }
        g.mqtt.publish(MQTT_LOG, msg)

#=====================================================================
# MyPrint
#---------------------------------------------------------------------
def MyPrint(str, use_system_log = True):
    # Get formatted date/time
    fdtm = DateAsString()

    # Print and flush
    flog = f"{fdtm} {str}"
    print(flog)
    sys.stdout.flush()

    if use_system_log:
        SystemLog(str)

def MyPrintNoSystemLog(str):
    MyPrint(str, False)

def DateAsString():
    # Get current date/time
    dt = datetime.datetime.now()

    # Format date/time string
    fdt = dt.strftime("%Y-%m-%d %H:%M:%S.%f")

    # Strip down to milliseconds
    fdtm = fdt[0 : len(fdt) - 3]

    return fdtm

#=====================================================================
# mqtt_handler
#---------------------------------------------------------------------
def mqtt_handler(topic, payload):
    try:
        if topic == MQTT_TRAFFIC_LIGHT:
            MyPrint(f"Receive MQTT: TRAFFIC_LIGHT")
        else:
            MyPrint(f"{g.app_name}: mqtt_handler: Unhandled topic {topic}")
    except Exception as e:
        exc_type, exc_obj, exc_tb = sys.exc_info()
        MyPrint(f"{g.app_name}: EXCEPTION mqtt_handler: {e} Line {exc_tb.tb_lineno}")

#=====================================================================
# set_colour
#----------------------------------------------------------------------
def set_colour(id, colour, sequence):
    if g.mqtt != None:
        msg = {
            "timestamp": DateAsString(),
            "client":    g.app_name,
            "id":        id,
            "colour":    colour,
            "sequence":  sequence,
        }
        g.mqtt.publish(MQTT_TRAFFIC_LIGHT, msg)
        MyPrint(f"Sent MQTT") 
        
#**********************************************************************
# run
#----------------------------------------------------------------------
def run(args=None):
    try:
        MyPrint(f"Set TL {g.version} start...")

        g.mqtt = Mqtt(g.app_name, MyPrint, None, None)

        if len(sys.argv) > 3:
            set_colour(sys.argv[1], sys.argv[2], sys.argv[3])
        else:
            set_colour(sys.argv[1], sys.argv[2], '')
        time.sleep(1)

    except Exception as err:
        MyPrint(f"EXCEPTION {err}")        
        
if __name__ == '__main__':
    run()

Make sure that the paho-mqtt library is installed:

bash
pip install "paho-mqtt<2.0.0"

To run the script:

bash
python3 tlset.py <traffic_light_id> <command> [<sequence>]

where:

  • <traffic_light_id> is one of following (labelled on base of traffic light):
    • trafficLightA
    • trafficLightB
    • trafficLightC
    • trafficLightD
  • <command> is one of:
    • red
    • yellow
    • green
    • all (all lights on)
    • off (all lights off)
    • sequence (uses additional parameter <sequence>)
  • <sequence>:
    • Optional parameter for use with "sequence" <command>
    • Double-quote delimited string
    • Consists of one or more "parts" delimited by commas
    • Each part has the format "<command>:<time>" where <command> is one of "red", "yellow", "green", "all" and "off" and <time> is the time in seconds
    • Automatically repeats until another <command> received

Sequence Examples:

Standard "red" -> "green" -> "yellow" (and then repeat) sequence:

bash
python3 tlset.py trafficLightA sequence "red:10.0,green:10.0,yellow:2.0"

Flashing "red":

bash
python3 tlset.py trafficLightA sequence "red:1.0,off:1.0"

All lights flashing:

bash
python3 tlset.py trafficLightA sequence "all:1.0,off:1.0"
