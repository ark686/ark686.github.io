1. Installing Client on Ubuntu ​

To install cfclient:

sudo apt install libxcb-xinerama0 libxcb-cursor0
pip3 install --break-system cfclient
1
2

2. Running Client ​

  1. Install antenna into USB port
  2. Connect controller via USB
  3. Turn on crazyflie
  4. Run cfclient
  5. Click on "Scan" and then click on "Connect"
  6. Select "Input device > Normal > Device > Input map > xbox360_mode1"

3. Flying ​

  1. Hold "R1" to launch and hover, release to land
  2. Forward = Left stick up
  3. Backward = Left stick down
  4. Strafe left = Left stick left
  5. Strafe right = Left stick right
  6. Up = Right stick up
  7. Down = Right stick down
  8. Rotate left = Right stick left
  9. Rotate right = Right stick right
