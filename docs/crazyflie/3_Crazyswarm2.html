2. Crazyflie Server ​

To launch the Crazyflie Server to control a Crazyflie using ROS2:

ros2 launch crazyflie_examples keyboard_velmux_launch.py

Can then use ROS2 teleop_twist_keyboard:

ros2 run teleop_twist_keyboard teleop_twist_keyboard

Use ‘t’ to take off, and ‘b’ to land. For the rest, use the instructions of the teleop package.

4.1 Takeoff Service ​

$ ros2 service call /cf231/takeoff crazyflie_interfaces/srv/Takeoff "{height: 0.3}"

4.2 Land Service ​

$ ros2 service call /cf231/land crazyflie_interfaces/srv/Land

4.3 Go To Service ​

$ ros2 service call /cf231/go_to  crazyflie_interfaces/srv/GoTo "{goal:{x: 0.0, y: 0.0, z: 1.0}}"
