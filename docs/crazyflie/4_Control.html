Crazyflie Control ​

ZIP file containing extended Crazyswarm2 and Python control program (2026-03-16):
crazyflie_control.zip

1. Run ​

  • Plug in Crazyflie radio
  • Turn on Crazyflie
. run.bash

2. control.py ​
