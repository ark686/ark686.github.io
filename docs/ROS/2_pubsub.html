Simple Publisher and Subscriber ​

Taken from https://docs.ros.org/en/jazzy/Tutorials/Beginner-Client-Libraries/Writing-A-Simple-Py-Publisher-And-Subscriber.html

1. Create ROS 2 Package ​

Create a new ROS 2 Package pubsub but no --node-name (can follow procedure in Build ROS 2 Packages).

2. Write the Publisher Node ​

2.1 Create pub.py ​

Create file ~/ros2_ws/src/pubsub/pubsub/pub.py with the following code:

python
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class MinimalPublisher(Node):
    def __init__(self):
        super().__init__('minimal_publisher')
        self.publisher_ = self.create_publisher(String, 'topic', 10)
        timer_period = 0.5  # seconds
        self.timer = self.create_timer(timer_period, self.timer_callback)
        self.i = 0

    def timer_callback(self):
        msg = String()
        msg.data = 'Hello World: %d' % self.i
        self.publisher_.publish(msg)
        self.get_logger().info('Publishing: "%s"' % msg.data)
        self.i += 1

def main(args=None):
    rclpy.init(args=args)
    minimal_publisher = MinimalPublisher()
    rclpy.spin(minimal_publisher)

    # Destroy the node explicitly
    # (optional - otherwise it will be done automatically
    # when the garbage collector destroys the node object)
    minimal_publisher.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()

2.2 Add Dependencies ​

Note that pub.py references two python libraries:

text
rclpy
std_msgs

Edit ~/ros_ws/src/pubsub/package.xml to add dependencies after the <license> line:

xml
<exec_depend>rclpy</exec_depend>
<exec_depend>std_msgs</exec_depend>

2.3 Add an Entry Point ​

Edit ~/ros_ws/src/pubsub/setup.py to modify the entry_points element:

python
entry_points={
    'console_scripts': [
        'talker = pubsub.pub:main',
    ],
},

3. Write the Subscriber Node ​

3.1 Create sub.py ​

Create file ~/ros2_ws/src/pubsub/pubsub/sub.py with the following code:

python
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class MinimalSubscriber(Node):
    def __init__(self):
        super().__init__('minimal_subscriber')
        self.subscription = self.create_subscription(
            String,
            'topic',
            self.listener_callback,
            10
        )
        self.subscription  # prevent unused variable warning

    def listener_callback(self, msg):
        self.get_logger().info('I heard: "%s"' % msg.data)

def main(args=None):
    rclpy.init(args=args)
    minimal_subscriber = MinimalSubscriber()
    rclpy.spin(minimal_subscriber)

    # Destroy the node explicitly
    # (optional - otherwise it will be done automatically
    # when the garbage collector destroys the node object)
    minimal_subscriber.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()

3.2 Add Dependencies ​

Note that sub.py references the same two python libraries as pub.py, so no further dependencies need to be added to ~/ros_ws/src/pubsub/package.xml.

3.3 Add an Entry Point ​

Edit ~/ros_ws/src/pubsub/setup.py to add another entry to the entry_points element:

python
entry_points={
    'console_scripts': [
        'talker = pubsub.pub:main',
        'listener = pubsub.sub.main',
    ],
},

4. Build and Run ​

In ~/ros2_ws, run:

bash
colcon build --symlink-install --packages-select pubsub

Open a new terminal:

bash
cd ~/ros2_ws
source install/setup.bash
ros2 run pubsub talker

Open another new terminal:

bash
cd ~/ros2_ws
source install/setup.bash
ros2 run pubsub listener

In the listener window, should see:

text
[INFO] [1767141185.398617673] [minimal_subscriber]: I heard: "Hello World: 290"
[INFO] [1767141185.898247360] [minimal_subscriber]: I heard: "Hello World: 291"
[INFO] [1767141186.399036550] [minimal_subscriber]: I heard: "Hello World: 292"
[INFO] [1767141186.898036462] [minimal_subscriber]: I heard: "Hello World: 293"

Enter Ctrl-C in each terminal to stop both nodes.
