2. Introduction ​

Gazebo is a collection of open source software libraries designed to simplify development of high-performance applications. The primary audience for Gazebo are robot developers, designers, and educators. We will be using Gazebo to simulate various “robots” including the TurtleBot 4. To make things a bit confusing:

  • The current “Gazebo” was initially called “Ignition” (there was a previous version called “Gazebo”)
  • In 2022, there was a trademark obstacle regarding the use of the name "Ignition", so they decided to rename the new version “Gazebo”
  • The original Gazebo is now referred to as “Gazebo Classic” (see Figure below)
  • Care must be taken when finding information/tutorials on the Internet as they may indicate “Gazebo” but there are actually using Gazebo Classic

Also complicating matters is that the version of Gazebo to use depends on the version of ROS 2 we are using (see Figure below). Since we are using “ROS 2 Jazzy”, we will be using “GZ Harmonic” (also works better with the TurtleBot 4s).
