ROS ​

Documentation on using Robot Operating System (ROS) version 2, Jazzy version.

Official link: https://docs.ros.org/en/jazzy/

1. Overview ​

The Robot Operating System (ROS) is a set of software libraries and tools for building robot applications. From drivers and state-of-the-art algorithms to powerful developer tools, ROS has the open source tools you need for your next robotics project.

ROS supports development in both Python and C++.

2. ROS Development ​

  • Migrate from Humble to Jazzy - Migration issues from ROS Humble to ROS Jazzy.
  • Build ROS 2 Packages - How to build ROS 2 packages.
  • Simple Publisher and Subscriber - Write simple ROS 2 Publisher and Subscriber nodes.
  • Python Launch Files - Write python launch files to launch more complex ROS 2 scenarios (e.g., multiple nodes, including other applications like rviz2 and rqt_graph)

3. Support Applications ​

Besides core ROS itself, two important supporting applications are:

  • RViz - ROS Visualization is an open-source 3D visualization tool within the Robot Operating System (ROS), for visualizing a robot's sensor data, state, and environment.
  • Gazebo - A 3D robotics simulator, to create and test robot applications in a virtual environment.

4. Description Formats ​

Two different formats are used to describe robots and the environment:

  • URDF - Unified Robot Description Format, an XML file format used to describe a robot's physical structure for use in RViz.
  • SDFormat - Simulation Description Format, sometimes abbreviated as SDF, is an XML format that describes objects and environments for robot simulators, visualization, and control and is used by Gazebo.

There are tools to translate URDF to SDFormat.
