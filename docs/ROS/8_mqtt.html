MQTT ​

1. References ​

  • MQTT: The Standard for IoT Messaging, https://mqtt.org
  • Beginners Guide To The MQTT Protocol: http://www.steves-internet-guide.com/mqtt
  • paho-mqtt, https://pypi.org/project/paho-mqtt

2. Introduction ​

There are some logistical issues trying to communicate between multiple ROS2 programs and/or robots at the same time from the same laptop or have a ROS2 program and/or robot communicate between multiple laptops using ROS2 due to the way the ROS2 implement network routing. To deal with this, a standard generic publisher/subscriber system called MQTT can be used in unison with ROS2.

3. MQTT ​

MQTT is a lightweight publish/subscribe messaging protocol designed for M2M (machine to machine) telemetry in low bandwidth environments [1] and [2]. It was designed by Andy Stanford-Clark (IBM) and Arlen Nipper in 1999 for connecting Oil Pipeline telemetry systems over satellite. Although it started as a proprietary protocol it was released Royalty free in 2010 and became an OASIS standard in 2014.

MQTT stands for MQ Telemetry Transport but previously was known as Message Queuing Telemetry Transport. MQTT is fast becoming one of the main protocols for IoT (internet of things) deployments.

3.1 MQTT Brokers/Servers ​

Note: The original term was broker but it has now been standardized as Server. You will see both terms used. There are many MQTT brokers available that you can use for testing and for real applications. There are free self hosted brokers , the most popular being Mosquitto and commercial ones like HiveMQ. Mosquitto is a free open source MQTT broker that runs on Windows and Linux, which we will be using in EE464.

To connect to an MQTT Broker (for an MQTT Client to publish and subscribe to topics), the following information is needed:

  • IP address of the MQTT Broker
  • Port of the MQTT Broker (default 1883)
  • Username/password for MQTT broker

3.2 MQTT Clients ​

A good open source Python MQTT client package for publishing and subscribing to MQTT is “paho-mqtt” [3].

To install the Python package:

pip install "paho-mqtt<2.0.0"

To import the package into your Python script:

python
from paho.mqtt import client as mqtt_client

3.3 MQTT Server IP ​

When you connect to an MQTT server, you need to specify the "IP" address of the server.

In the Python examples in Sections 3.4 and 3.5, three examples are listed (two commented out):

python
    #mqttIp = '127.0.0.1'
    #mqttIp = '192.168.129.60'
    mqttIp = 'ece2.arkcom.ca'

  • mqttIp = '127.0.0.1' is for using the MQTT server on the local computer (when all the programs are on the same computer)
  • mqttIp = '192.168.129.60' is for using the MQTT server on a specific computer on the local area network, allows communications between programs on different computers/robots on the same local network
  • mqttIp = 'ece2.arkcom.ca' is for using an MQTT server set up on the public Internet, allows communications between programs on different/computers anywhere on the Internet as long as they have Internet access

All of the laptops in 2C95 and Hardy Lab should have an MQTT server installed and running.

The Internet server ece2.arkcom.ca should have an MQTT server installed and running. Note that the delay using this MQTT server will be somewhat longer than on the local machine or computer on the same local network.

3.4 MQTT Log Server ​

Example of a central Log Server (printing out logs from a number of programs) using MQTT:

python
######################################################################
# File: mqtt_log_server.py
#---------------------------------------------------------------------
import random
import time
import sys
import json
import datetime

from paho.mqtt import client as mqtt_client

class g:
    name = 'ee464_log_server'
    mqttClientId = ''
    mqtt = None
    mqttUsername = 'ee464'
    mqttPassword = 'badapple2025'
    #mqttIp = '127.0.0.1'
    #mqttIp = '192.168.129.60'
    mqttIp = 'ece2.arkcom.ca'
    mqttPort = 1883

MQTT_LOG = 'log'

MQTT_TOPICS = [
    (MQTT_LOG, 0),
]

######################################################################
# MyPrint
#---------------------------------------------------------------------
def MyPrint(str):
    fdtm = DateAsString()
    print(f"{fdtm} {str}\r")
    sys.stdout.flush()

def DateAsString():
    # Get current date/time
    dt = datetime.datetime.now()

    # Format date/time string
    fdt = dt.strftime("%Y-%m-%d %H:%M:%S.%f")

    # Strip down to milliseconds
    fdtm = fdt[0 : len(fdt) - 3]

    return fdtm

######################################################################
# MqttConfigure
#---------------------------------------------------------------------
def MqttConfigure():
    def on_connect(client, userdata, flags, rc):
        if rc == 0:
            MyPrint(f"MqttConfigure: Connected to MQTT Broker ({g.mqttClientId})")
        else:
            MyPrint(f"MqttConfigure: Failed to connect, return code {rc}")

    def on_message(client, userdata, msg):
        try:
            if not msg.payload:
                payload = None
            else:
                payload = json.loads(msg.payload)

            if msg.topic == MQTT_LOG:
                MyPrint(f"MQTT_LOG:{payload}")
                pass
            else:
                MyPrint(f"Mqtt.on_message Unhandled topic {msg.topic}")
        except Exception as e:
            exc_type, exc_obj, exc_tb = sys.exc_info()
            MyPrint(f"MqttConfigure.on_message: {e} Line {exc_tb.tb_lineno}")
                    
    try:
        g.mqttClientId = f"{g.name}_{random.randint(1000, 9999)}"
        g.mqtt = mqtt_client.Client(g.mqttClientId)
        g.mqtt.username_pw_set(g.mqttUsername, g.mqttPassword)
        g.mqtt.on_connect = on_connect
        g.mqtt.connect(g.mqttIp, g.mqttPort)
        g.mqtt.subscribe(MQTT_TOPICS)
        g.mqtt.on_message = on_message
        g.mqtt.loop_start()
    except Exception as e:
        exc_type, exc_obj, exc_tb = sys.exc_info()
        MyPrint(f"MqttConfigure: {e} Line {exc_tb.tb_lineno}")

######################################################################
# main
#---------------------------------------------------------------------
def main(args=None):
    MqttConfigure()
    while True:
        time.sleep(0.1)

if __name__ == "__main__":
    main()

3.5 MQTT Log Sender ​

Example of using MQTT to send logs to a central MQTT Log Server:

python
######################################################################
# File: mqtt_log_sender.py
#---------------------------------------------------------------------
import time
import sys
import json
import random
import datetime
from paho.mqtt import client as mqtt_client

class g:
    name = 'mqtt_log_send'
    mqttClientId = ''
    mqtt = None
    mqttUsername = 'ee464'
    mqttPassword = 'badapple2025'
    #mqttIp = '127.0.0.1'
    #mqttIp = '192.168.129.60'
    mqttIp = 'ece2.arkcom.ca'
    mqttPort = 1883

MQTT_ORDER_DELIVERED = 'order_delivered'
MQTT_LOG = 'log'

######################################################################
# MyPrint
#---------------------------------------------------------------------
def MyPrint(str):
    fdtm = DateAsString()
    print(f"{fdtm} {str}\r")
    sys.stdout.flush()

def DateAsString():
    # Get current date/time
    dt = datetime.datetime.now()

    # Format date/time string
    fdt = dt.strftime("%Y-%m-%d %H:%M:%S.%f")

    # Strip down to milliseconds
    fdtm = fdt[0 : len(fdt) - 3]

    return fdtm

######################################################################
# MqttConfigure
#---------------------------------------------------------------------
def MqttConfigure():
    def on_connect(client, userdata, flags, rc):
        if rc == 0:
            MyPrint(f"MqttConfigure: Connected to MQTT Broker ({g.mqttClientId})")
        else:
            MyPrint(f"MqttConfigure: Failed to connect, return code {rc}")

    def on_message(client, userdata, msg):
        try:
            if not msg.payload:
                payload = None
            else:
                payload = json.loads(msg.payload)

            MyPrint(f"MqttConfigure: msg.topic={msg.topic}")
            if msg.topic.startswith('MQTT_COMMAND'):
                pass
                
        except Exception as inst:
            MyPrint(f"Mqtt.on_message EXCEPTION {inst}")
                    
    try:
        g.mqttClientId = f"ee464_order_{random.randint(1000, 9999)}"
        g.mqtt = mqtt_client.Client(g.mqttClientId)
        g.mqtt.username_pw_set(g.mqttUsername, g.mqttPassword)
        g.mqtt.on_connect = on_connect
        g.mqtt.connect(g.mqttIp, g.mqttPort)
                
        #g.mqtt.subscribe(MQTT_TOPICS)
        g.mqtt.on_message = on_message
        g.mqtt.loop_start()
    except Exception as e:
        exc_type, exc_obj, exc_tb = sys.exc_info()
        MyPrint(f"MqttConfigure: {e} Line {exc_tb.tb_lineno}")

#**********************************************************************
# SystemLog
#----------------------------------------------------------------------
def SystemLog(log):
    try:
        if g.mqtt == None:
            MyPrint(f"SystemLog: Aborted log '{log}'")
            return
        msg = {
            "timestamp": DateAsString(),
            "client":    g.name,
            "log":       log
        }
        result = g.mqtt.publish(MQTT_LOG, json.dumps(msg))
        status = result[0]
        if status == 0:
            MyPrint(f"SystemLog: Sent log '{log}'")
        else:
            MyPrint("SystemLog: Failed to send log")
    except Exception as e:
        exc_type, exc_obj, exc_tb = sys.exc_info()
        MyPrint(f"MqttConfigure: {e} Line {exc_tb.tb_lineno}")

#**********************************************************************
# Main
#----------------------------------------------------------------------
def main(args=None):
    try:
        MqttConfigure()
        count = 0
        while True:
            time.sleep(1.0)
            count = count + 1
            SystemLog(f"Log #{count}")
    except Exception as e:
        exc_type, exc_obj, exc_tb = sys.exc_info()
        MyPrint(f"An exception occurred: {e} Line {exc_tb.tb_lineno}")
        
if __name__ == "__main__":
    main()
