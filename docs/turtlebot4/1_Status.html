Turtlebot4 Status ​

Turtlebot4 Status Page: https://ece2.arkcom.ca/tb4

For laptops (Text colour indicates reporting status, BLACK is active, RED is not reporting):

  • TIMESTAMP is time of last reporting
  • VERSION is version of control and monitoring software
  • ID is the identifier of the laptop
  • IP is the IP address of the laptop
  • UPTIME is the time since last boot
  • USERS are the currently logged in users
  • OS is Ubuntu version
  • ROS is ROS version
  • TB4 is index of associated Turtlebot4
  • TB4IP is IP address of associated Turtlebot4

For Turtlebot4s (Text colour indicates reporting status, BLACK is active, RED is not reporting):

  • TIMESTAMP is time of last reporting
  • VERSION is version of control and monitoring software
  • ID is the identifier of the Turtlebot4
  • IP is the IP address of the Turtlebot4
  • UPTIME is the time since last boot
  • USERS are the currently logged in users
  • OS is Ubuntu version
  • ROS is ROS version
  • BATTERY is Turtlebot4 battery percentage
  • LIDAR is indication if LIDAR is operation ('lidar' or 'no-lidar')
